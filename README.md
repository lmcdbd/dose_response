# OVERVIEW

This repository contains the R scripts used in the manuscript:

**"Plate-based ISD-SPE enables dual proteome-secretome dose-response profiling of TLR signalling in iPSC-derived macrophages."**

- `curve_fitting.R`: Performs dose–response curve fitting for proteome and secretome data.
- `curve_plotting.R`: Generates dose–response plots from the fitted curve results.
